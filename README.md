# vlcjideo
Spring boot application with JavaFX and vlcj to display RTSP videos

```
curl -X POST -H 'Content-Type: application/json' -i http://localhost:1971/api/v1/video/set --data '{
  "url":"https://www.youtube.com/watch?v=rtC84CMVILw"
}'
```
